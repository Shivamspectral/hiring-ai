## рџ“Љ Candidate Intelligence

Every candidate in TalentLens gets a full AI-generated profile:

## вљ–пёЏ Built-in Bias Detection

TalentLens scans every job description for:
- **Gendered language** вЂ” "rockstar", "ninja", "manpower", "chairman"
- **Age bias** вЂ” "digital native", "young and hungry", "recent graduate"
- **Exclusionary phrases** вЂ” "no gaps in employment", "must work long hours"
- **Cultural bias** вЂ” overly narrow culture-fit language

For each issue, AI provides the original phrase, an inclusive replacement, and the reason for the change. It also generates a fully rewritten inclusive JD.

## вљ™пёЏ Setup & Installation

### Prerequisites
- Python 3.10+
- Node.js 18+
- Accounts on: [Groq](https://console.groq.com), [Supabase](https://supabase.com), [Pinecone](https://pinecone.io), [Google AI Studio](https://aistudio.google.com)

### 1. Clone the repo
```bash
git clone https://github.com/Shivamspectral/talentlens.git
cd talentlens
```

### 2. Backend setup
```bash
cd backend
pip install -r requirements.txt
```

Create `.env` in `/backend`:
```env
GROQ_API_KEY=your_groq_key
GEMINI_API_KEY=your_gemini_key
PINECONE_API_KEY=your_pinecone_key
PINECONE_INDEX=hiring-pipeline
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_KEY=your_supabase_anon_key
FRONTEND_URL=https://talentlens-ai-hiring.vercel.app
```

Run Supabase SQL setup:
```sql
create table if not exists jobs (
  id uuid default gen_random_uuid() primary key,
  title text, description text, requirements jsonb,
  created_at timestamp default now()
);
create table if not exists candidates (
  id uuid default gen_random_uuid() primary key,
  job_id uuid references jobs(id),
  name text, email text, phone text,
  resume_text text, parsed_data jsonb,
  match_score integer, match_explanation jsonb,
  status text default 'pending',
  created_at timestamp default now()
);
create table if not exists ai_cache (
  id uuid default gen_random_uuid() primary key,
  cache_key text unique not null,
  namespace text, result jsonb,
  created_at timestamp default now()
);
```

Start backend:
```bash
uvicorn main:app --reload --port 8000
```

### 3. Frontend setup
```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
